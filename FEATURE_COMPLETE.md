## 如何使用

### 网页版
```bash
cd web
npm install
npm run dev
```

### 微信小程序
```bash
cd miniprogram
npm install
npm run dev:mp-weixin
```

### 鸿蒙版
```bash
cd harmony
npm install
# 使用DevEco Studio打开项目
```

### Docker部署
```bash
docker-compose up -d
```
