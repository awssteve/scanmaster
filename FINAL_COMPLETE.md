## 如何使用

### 网页版
```bash
cd web
npm install
npm run dev
```
访问：http://localhost:3000

### 微信小程序
```bash
cd miniprogram
npm install
npm run dev:mp-weixin
```

### 鸿蒙版
```bash
cd harmony
npm install
# 使用DevEco Studio打开项目
```

### Docker部署（全平台）
```bash
docker-compose up -d
```
访问：http://localhost

## 风险和应对

### 1. 技术风险
**风险：** OCR准确率不够高
**应对：** 持续优化模型，收集用户反馈

### 2. 成本风险
**风险：** GPU服务器成本高
**应对：** 前期使用本地4090，后期根据用户量上GPU云服务器

### 3. 竞争风险
**风险：** 扫描全能王竞争激烈
**应对：** 差异化定位（免费+教育场景），快速占领细分市场

### 4. 用户获取风险
**风险：** 用户增长缓慢
**应对：** 多渠道推广，利用高校平台资源
